El objetivo de esta tarea fue crear una página web utilizando 
<strong>GitHub Pages</strong>, aplicando conocimientos de <strong>HTML, CSS y JavaScript</strong>. 
Cada componente de la web (imágenes, estilos, scripts...) debía estar alojado en el 
propio repositorio de GitHub. Además, la página debe incluír funcionalidades interactivas con JavaScript, 
como intercambio de imágenes, modificación de texto por el usuario y un objeto en movimiento, 
así como una fuente distintiva y enlaces accesibles. La página web que yo he creado es sobre viajes, 
puedes verla en el siguiente enlace:
